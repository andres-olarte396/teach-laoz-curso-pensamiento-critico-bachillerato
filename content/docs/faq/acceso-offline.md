# Acceso Offline

Sabemos que el aprendizaje no siempre ocurre donde hay una conexión estable a internet.

## Estado actual

Actualmente, Teach LAOZ requiere una conexión a internet activa para:

- Registrar tu progreso en tiempo real.
- Validar tus sesiones de usuario.
- Cargar simulaciones interactivas pesadas.

## Visualización sin conexión (Beta)

Estamos trabajando en una versión descargable para nuestra aplicación móvil que permitirá:

1. **Descargar lecciones**: Podrás bajar videos y textos para estudiarlos en aviones o zonas remotas.
2. **Sincronización diferida**: Una vez que vuelvas a tener conexión, tu progreso se sincronizará automáticamente con el servidor.

> [!NOTE]
> Por ahora, te recomendamos descargar los archivos PDF de recursos (guías, ejercicios) que se encuentran en la barra de "Material Descargable" de cada curso para estudiarlos offline.
