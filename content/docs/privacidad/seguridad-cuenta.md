# Seguridad de Cuenta

Tu cuenta es la llave a tu conocimiento. Sigue estas recomendaciones para mantenerla protegida.

## Contraseñas Robustas

- Usa una combinación de letras (mayúsculas y minúsculas), números y símbolos.
- Evita usar información personal como fechas de nacimiento o nombres de mascotas.
- Cambia tu contraseña periódicamente desde los ajustes de perfil.

## Autenticación de Dos Factores (2FA)

Recomendamos encarecidamente activar el 2FA. Cada vez que inicies sesión desde un dispositivo nuevo, se te pedirá un código enviado a tu aplicación de autenticación (como Google Authenticator o Authy).

## Sesiones Activas

En tu perfil puedes ver una lista de todos los dispositivos donde tienes una sesión iniciada. Si ves algo sospechoso, puedes cerrar todas las sesiones con un solo clic.

> [!CAUTION]
> Teach LAOZ nunca te pedirá tu contraseña por correo electrónico ni por redes sociales. Si recibes una solicitud así, repórtala inmediatamente.
