# Realizar Ejercicios

La práctica es la clave del dominio técnico. En Teach LAOZ, muchos módulos incluyen ejercicios prácticos integrados.

## Tipos de Ejercicios

1. **Cuestionarios (Tests)**: Preguntas de opción múltiple para validar conceptos teóricos.
2. **Ejercicios Técnicos**: Consisten en resolver un problema específico relacionado con el módulo (ej. corregir una animación SVG o configurar un contenedor Docker).

## Cómo acceder y resolver

- **Botón Hacer Ejercicios**: Al final de muchas lecciones, verás un botón verde resaltado.
- **Rúbricas**: Cada ejercicio incluye criterios de evaluación para que sepas exactamente qué se espera de tu solución.

## Revisión y Feedback

Una vez enviado, recibirás feedback automático o podrás comparar tu solución con la guía del experto proporcionada en la misma sección.

> [!NOTE]
> Algunos ejercicios requieren que descargues un repositorio de GitHub para trabajar localmente en tu computadora.
