# Sistema de Notas

Nunca pierdas una idea brillante. Nuestro sistema de notas te permite capturar pensamientos importantes mientras estudias.

## Cómo usar las notas

1. **Panel de Notas**: Mientras visualizas una lección, busca el icono de bloc de notas en la interfaz.
2. **Escritura en tiempo real**: Tus notas se guardan automáticamente mientras escribes.
3. **Sincronización**: Podrás acceder a todas tus notas desde tu perfil, organizadas por curso y lección.

## Formato de Notas

Puedes usar formato básico (Markdown) para resaltar partes de tus notas, como **negritas** o bloques de `código`.

> [!TIP]
> Te recomendamos anotar las marcas de tiempo (ej. "Minuto 05:20") si quieres recordar una explicación específica de un audio o video.
