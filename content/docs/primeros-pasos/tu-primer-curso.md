# Tu primer curso

¿Listo para aprender? Sigue estos pasos para inscribirte y comenzar tu primera lección.

## Cómo empezar

1. **Busca un curso**: Ve a la sección [Explorar Cursos](/courses) y encuentra un tema que te interese.
2. **Ver detalles**: Haz clic en la tarjeta del curso para leer el resumen, ver el nivel de dificultad y el autor.
3. **Iniciar lección**: Una vez dentro del curso, haz clic en la primera lección del menú lateral del curso para comenzar a ver el contenido.

## Interfaz de Aprendizaje

Dentro de cada curso disfrutarás de:

- **Contenido Multiformato**: Textos claros, imágenes y, en algunos casos, audios y videos.
- **Simulaciones**: Algunos cursos técnicos incluyen elementos interactivos para practicar.
- **Barra de navegación**: Muévete fácilmente entre módulos y lecciones desde el lateral derecho.

> [!NOTE]
> No te preocupes por guardar tu progreso; el sistema lo hace automáticamente para que nunca pierdas el hilo.
