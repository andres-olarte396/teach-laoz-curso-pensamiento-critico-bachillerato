# Evaluación: Método Científico

1.  **¿Cuál es el "superpoder" del método científico?**
    - a) Tener siempre la razón.
    - b) La Falsabilidad (intentar probar que estás equivocado).
    - c) Usar batas blancas.
2.  **Si tu hipótesis falla en el experimento, ¿qué haces?**
    - a) Llorar.
    - b) Cambiar los datos para que encajen.
    - c) Crear una nueva hipótesis y volver a probar.

**(Respuestas Correctas)**
1-b, 2-c
