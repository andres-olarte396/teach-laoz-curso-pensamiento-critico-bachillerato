## � Descripción del Curso

Este repositorio alberga los recursos educativos para el curso **Pensamiento Crítico para Bachillerato**. El objetivo principal es fortalecer las habilidades analíticas de los estudiantes, permitiéndoles evaluar información de manera objetiva, identificar manipulaciones y tomar decisiones fundamentadas.

El plan de estudios se estructura en 5 módulos progresivos:

1.  **Sesgos Cognitivos y Heurísticos:** Comprender cómo funciona nuestro cerebro y sus "atajos".
2.  **Falacias Lógicas:** Identificar errores en la argumentación propia y ajena.
3.  **Verificación y Fake News:** Herramientas para detectar desinformación y aplicar el método científico.
4.  **Ciudadanía Digital:** Privacidad, algoritmos y comportamiento ético en línea.
5.  **Resolución de Conflictos:** Inteligencia emocional y estrategias para el diálogo constructivo.

Incluye guiones, videos, simulaciones interactivas, evaluaciones y rúbricas.
